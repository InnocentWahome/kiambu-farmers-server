# Eveenti - (server)

> Ticketing Application

## Quick Installation

```bash
### Clone the git repository
git clone https://github.com/Eveenti-Ticketing/adonis-server.git

### Setup environment variables
cp .env.example .env

### Install project dependencies
npm install

### Initialize development server
npm run dev

### Run production
npm start
```
